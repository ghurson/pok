\NV\Theme\CustomWalkerComments

This is our custom comment walker. Since comments can have complex parent-child relationships, we utilize Walker to handle output instead of a more traditional loop.

This modifies the default display_element() function to "flatten" any replies that exceed the max_depth.

This function is taken from the Walker_Comment class in WordPress core.
