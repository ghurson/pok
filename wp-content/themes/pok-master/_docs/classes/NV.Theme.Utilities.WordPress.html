is_tax_archive()

is_tax_archive() : boolean

Checks whether a page is a taxonomy archive (of ANY type). This is more broad than WordPress' built-in is_tax() since this considers both 'tag' and 'category' to be general taxonomies.

Returns

boolean —

True is the current page is a taxonomy archive of any type.

is_post_type_archive() : boolean

Checks whether a page is is a post archive. Unlike WordPress' built-in is_post_type_archive() function, this returns true for the built-in 'post' archive / home.

Returns

boolean —

get_dashboard_widget_options(string  $widget_id = '', string  $option = '', mixed  $default = false) : array

Gets the options for a widget of the specified name.
