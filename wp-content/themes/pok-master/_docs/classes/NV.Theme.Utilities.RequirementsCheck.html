check_for_admin()

check_for_admin()

Generally, you shouldn't call this method directly. Instead, use:

add_action( 'admin_notices', array( &$this, 'show_admin_error' )

This will show a message on the WordPress admin if the minimum requirements are not met. This gives admins the ability to shut down the theme if needed by not breaking the admin.
